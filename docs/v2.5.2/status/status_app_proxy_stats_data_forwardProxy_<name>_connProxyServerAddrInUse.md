## /status/app/proxy/stats/data/forwardProxy/<name>

Use

Use to display information and statistics about a forward proxy.

### Parameters

name: Name of forward proxy object

#### Data Type

subtree

#### Default Data Value

0

### Request Methods

GET

### Related

d_Proxy_Commands)
