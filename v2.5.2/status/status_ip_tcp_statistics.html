/status/ip/tcp/statistics¶

Use

Use to view the TCP protocol statistics.

Data Key¶

The data key contains the statistics.

Data Type¶

string

Default Data Value¶

“”

Request Methods¶

GET

[REST API Reference - /status/ip](https://docs.lineratesystems.com/093Release_ 2.5/250REST_API_Reference_Guide/status/ip)

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/ip/tcp/statistics
