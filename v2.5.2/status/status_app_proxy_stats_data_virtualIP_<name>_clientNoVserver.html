/status/app/proxy/stats/data/virtualIP/¶

Use

Use to display snapshot of statistics related to the virtual IP. The statistics only show the termination side of connections (from clients).

Parameters¶

name: Name of virtual IP

Data Type¶

subtree

Default Data Value¶

0

Request Methods¶

GET

Related¶

[CLI Reference - Show Virtual IP Commands](https://docs.lineratesystems.com/09 3Release_2.5/200CLI_Reference_Guide/Exec_Commands/Show_Commands/Show_Virtual_I P_Commands)
