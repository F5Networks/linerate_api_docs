/status/app/proxy/realServerSummary/routes¶

Use

Use to view routing information that the system uses to access the real server. This is not user configurable.

Data Key¶

The data key contains the routing information.

Data Type¶

string

Default Data Value¶

“”

Request Methods¶

GET
