/status/app/proxy/stats/data/realServer//httpServerRequestRateLimit50¶

3PerSec1MinAvg

Use

Use to view the 1-minute average of the number of requests responded to with 503 due to rate limit exceeded.

Parameters¶

name: Name of real server object

Data Key¶

The data key contains the number of requests.

Data Type¶

uint64

Default Data Value¶

0

Request Methods¶

GET

Related¶

CLI Reference - [Show Real Server Commands](https://docs.lineratesystems.com/0 93Release_2.5/200CLI_Reference_Guide/Exec_Commands/Show_Commands/Show_Real_Ser ver_Commands)

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/app/proxy/stat s/data/realServer/rs-01-01/httpServerRequestRateLimit503PerSec1MinAvg
