/status/app/proxy/stats/data/global/virtualServer/httpProxyClientRespInitLa¶

tency1MinAvg

Use

Use to view the 1-minute moving average of the time between receiving the first byte of a request from a client and sending the first byte of the response for that request to the client. The moving average is computed using latency samples taken every five seconds.

Data Key¶

Data Type¶

double

Request Methods¶

GET
