/status/app/licensing/rateLimit/burstDuration

Use

Use to view the burst duration, which acts as a multiplier for each licensed rate limit, that the system is using. The <burst_duration> x each rate limit sets the maximum number of HTTP requests, TCP connections, and Mb per second the system can handle in a burst above the rate limit. 

Data Key

The data key contains the burst duration multiplier.

Data Type

uint64

Default Data Value

0

Request Methods

GET
