/config/app/health/monitor/<name>/httpOptions/response

Use

The nodes below this one configure settings for the HTTP responses.

Set the service type for the health monitor in the type node.

Parameters

name: Name of health monitor

Data Type

subtree

Request Methods

GET

Related

REST API Reference - /config/app/health/monitor
