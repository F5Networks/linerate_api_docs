/config/app/proxy/virtualServer/<name>/realServer/<rs_name>

Use

Use to associate a real server with the virtual server.

Parameters

name: Name of virtual server object
rs_name: Name of real server to attach—Must use only a limited character set.

Data Key

The data key must match the name of the real server.

Data Type

string

Default Data Value

""

Default Allowed

False

Request Methods

GET, POST, PUT, DELETE

Related

REST API Reference - /config/app/proxy/virtualServer

CLI Reference - Virtual Server Mode Commands

Examples

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/config/app/proxy/virtualServer/vs1/realServer/rs1
