/status/app/proxy/stats/data/forwardProxy/<name>/licenseProxyRequestRateLimitExceededPerSec1MinAvg

Use

Use to view the 1-minute average of the number of requests responded to with 503 due to license rate limit exceeded.

Parameters

name: Name of forward proxy object

Data Key

The data key contains the number of requests.

Data Type

uint64

Default Data Value

0

Request Methods

GET

Related

CLI Reference - Show Forward Proxy Commands

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/app/proxy/stats/data/forwardProxy/fp1/licenseProxyRequestRateLimitExceededPerSec1MinAvg
