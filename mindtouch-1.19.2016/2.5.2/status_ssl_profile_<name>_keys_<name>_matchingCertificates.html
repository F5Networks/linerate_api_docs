/status/ssl/profile/<name>/keys/<name>/matchingCertificates

Use

Use the view the names of key's matching certificates.

Parameters

name: Name of SSL profile
name: Name of keys object

Data Key

The data key contains the names of certificates that match the key that are attached to the SSL profile or 0 if no matching certificates are attached.

Data Type

json

Default Data Value

0

Request Methods

GET

Related

REST API Reference - /status/ssl

CLI Reference - Show SSL Commands

Examples

GET

curl -b cookie.jar -k https://host-128:8443/lrs/api/v1.0/status/ssl/profile/sp-01/keys/name/matchingCertificates
