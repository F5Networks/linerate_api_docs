/config/syslog/servers/<server_name>/port

Use

Configure a different port on which this remote logging server receives messages.

Parameters

server_name: name of server

Data Key

The data key contains the port.

Data Type

uint32

Default Data Value

514

Default Allowed

True

Request Methods

GET, PUT

Related

REST API Reference - /config/syslog

CLI Reference Guide - Logging Mode Commands

Examples

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/config/syslog/servers/name/port
