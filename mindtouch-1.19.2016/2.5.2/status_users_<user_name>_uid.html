/status/users/<user_name>/uid

Use

Use to view the user's unique user ID.

Parameters

user_name: Name of user account

Data Key

The data key contains the UID.

Data Type

uint32

Default Data Value

0

Default Allowed

False

Request Methods

GET

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/users/admin/uid
