/config/app/proxy/realServerBase/<base_name>/serviceHttp

Use

The nodes below this one configure settings for the HTTP service type.

Set the service type for the real server in serviceType.

Parameters

base_name: Name of real server base

Data Type

subtree

Request Methods

GET

Related

REST API Reference - /config/app/proxy/realServerBase

CLI Reference - Real Server Mode Commands
