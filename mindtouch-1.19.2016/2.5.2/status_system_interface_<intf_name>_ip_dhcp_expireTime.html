/status/system/interface/<intf_name>/ip/dhcp/expireTime

Use

If DHCP is enabled, use see when the DHCP-assigned IP address expires, in UNIX time.

Parameters

intf_name: Name of interface object

Data Key

The data key contains the expiration time, in UNIX time.

Data Type

uint64

Default Data Value

0

Request Methods

GET

Related

REST API Reference - dhcp

CLI Reference - ip address dhcp

 

Examples

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/system/interface/em1/ip/dhcp/expireTime
