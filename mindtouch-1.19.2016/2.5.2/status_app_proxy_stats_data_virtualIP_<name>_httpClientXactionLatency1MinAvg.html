/status/app/proxy/stats/data/virtualIP/<name>/httpClientXactionLatency1MinAvg

Use

Use to view the 1-minute average of the average time between receiving the first byte of the HTTP request from a client and sending the last byte of the response for that request to the client.

Parameters

name: Name of virtual IP

Data Key

The data key contains the time in seconds.

Data Type

double

Default Data Value

0

Request Methods

GET

Related

CLI Reference - Show Virtual IP Commands

Examples

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/app/proxy/stats/data/virtualIP/vip1/httpClientXactionLatency1MinAvg
