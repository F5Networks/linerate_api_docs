/status/system/interface/<intf_name>/ip/addrMask/<ipAddr>

Use

Use to view the configured IP address for the interface.

Parameters

intf_name: Name of interface
ipAddr: IPv4 or IPv6 address—Must use standard F5® LineRate® IP address format.

Data Key

The IP address within data key must match the IP address in the ipAddr node name.

Data Type

ip-addr-with-mask

Default Allowed

False

Request Methods

GET

Related

REST API Reference - /config/system/interface

CLI Reference - Show Interfaces Commands

Examples

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/system/interface/em1/ip/addrMask/201.1.2.3
