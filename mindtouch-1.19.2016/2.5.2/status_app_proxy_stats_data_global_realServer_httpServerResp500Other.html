/status/app/proxy/stats/data/global/realServer/httpServerResp500Other

Use

Use to view the number of HTTP responses with other 5xx response codes received from the servers.

Data Key

The data key contains the number of responses.

Data Type

uint64

Default Data Value

0

Request Methods

GET
