/status/app/proxy/stats/data/virtualIP/<name>/httpInternalResp304

Use

Use to view the number of HTTP responses with response code 304 sent to the clients.

Parameters

name: Name of virtual IP

Data Key

The data key contains the number of responses.

Data Type

uint64

Default Data Value

0

Request Methods

GET

Related

CLI Reference - Show Virtual IP Commands

Examples

GET

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/app/proxy/stats/data/virtualIP/vip1/httpInternalResp304
