/config/app/proxy/virtualServer/<name>/virtualIP

The nodes below this one perform the functions described in this function overview.

Use

Use to attach a virtual IP to the virtual server. You normally attach one virtual IP to a virtual server. See virtualIP.

Parameters

name: Name of virtual server object

Data Type

subtree

Request Methods

GET

Related

REST API Reference - /config/app/proxy/virtualServer

CLI Reference - Virtual Server Mode Commands
