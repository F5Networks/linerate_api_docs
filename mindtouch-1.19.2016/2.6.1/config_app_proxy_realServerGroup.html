/config/app/proxy/realServerGroup

The nodes below this one perform the functions described in this function overview.

Create a real server group.

Use

Use to create real server groups to make configuration more efficient. You can attach a real server group to a virtual server. Group real servers based on those served by the same virtual IP and virtual server.

Data Type

subtree

Request Methods

GET

Related

CLI Reference - Real Server Mode Commands
