/status/restServer/details/allow/from/<address>

Use

Use to view the configuration to permit connections from the specified IP addresses to the REST server.

Parameters

address: REST client address—IPv4 or IPv6 address with subnet mask. Must use standard F5® LineRate® IP address format and modified CIDR notation. For example: 192.0.2.1 24. Do not use a slash before the subnet mask; use a space instead.

A node called "any" is the default. This lets the REST server accept connections from both IPv4 and IPv6 clients and overrides any "alllow from" addresses configured.

Data Key

The data key must match the IP address.

Data Type

string

Default Data Value

""

Request Methods

GET
