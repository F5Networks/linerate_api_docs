/status/syslog/filters/<filter_name>/levels/<level>

Use

Use to view the priority/severity of the messages configured for this filter.

Parameters

filter_name: Name of filter
level: Filter level

Data Key

The data key contains the filter level or range.

Data Type

string

Data Values

emerg (0)

alert (1)

crit (2)

err (3)

warning (4)

notice (5)

info (6)

debug (7)

Default Data Value

""

Default Allowed

False

Request Methods

GET, POST, PUT, DELETE

Related

REST API Reference - /config/syslog
