/status/syslog/servers/<server_name>/protocol</server_name>

Use

Use to view the protocol configured for this remote logging server.

Parameters

server_name: Name of server

Data Key

The data key contains protocol, either UDP or TCP.

Data Type

string

Default Data Value

"" (UDP)

Default Allowed

True

Request Methods

GET, PUT

Related

REST API Reference - /config/syslog

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/syslog/servers/name/protocol
