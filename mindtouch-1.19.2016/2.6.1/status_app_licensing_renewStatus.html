/status/app/licensing/renewStatus

Use

Use to view the state of the config/licensing/activationMode REST node or activation auto in the CLI.

Data Key

The data key contains one of the following:

  • INFO: Automatic license renewal enabled.
  • WARNING: Automatic license renewal is disabled. Set licensing activation auto to enable.

Data Type

string

Default Data Value

""

Request Methods

GET

Related

REST API Reference - licensing

CLI Reference - Show Licensing Commands

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/status/app/licensing/renewStatus
