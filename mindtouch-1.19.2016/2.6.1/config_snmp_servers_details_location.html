/config/snmp/servers/details/location

Use

Use to configure information to identify the system's location.

Data Key

The data key contains the location information.

Data Type

string

Default Data Value

""

Default Allowed

True

Request Methods

GET, PUT

Related

CLI Reference - SNMP Mode Commands

curl -b cookie.jar -k https://10.1.2.3:8443/lrs/api/v1.0/config/snmp/servers/name/location
