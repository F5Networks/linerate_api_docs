/config/app/proxy/virtualIP/<name>/ipFilter

Use

Use to associate an IP filter list with the virtual IP.

Set up the profile using the ip/filterList nodes.

Parameters

name: Name of virtual IP

Data Key

The data key contains the name of the IP filter list.

Note: When the default key is set to true, it means the object is set to its default. In this case, the system works as follows:

  • On a PUT operation, the system ignores the data key. (Applicable only to /config nodes.)
  • On a GET operation, the response only contains "default": True and does not contain the data key.
  • If the object has a base, it inherits its setting from its base.

See Setting Objects to Their Default.

Data Type

string

Default Data Value

""

An object uses the default data value when one of the following is true:

  • For objects without bases—When the object's default key is true.
  • For objects with bases—When the object's default key is true, and when the default key is true for all of the object's bases.

Default Allowed

True

Request Methods

GET, PUT
