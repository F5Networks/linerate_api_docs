/exec/debug/app/proxy/cpuProfile¶

Use

Enable CPU profiling for the proxy. Create CPU profile recordings in /home/data/linerate/profile. Only F5 can properly interpret these recordings. This command should only be used when directed by F5 support.

Caution: Enabling debug functions may have a significant impact on system performance. Use only with extreme caution on a production system. Debug recordings take disk space, so leaving a debug function enabled may use all available disk space.

Data Key¶

The data key contains one of the values in the Data Values below.

Data Type¶

uint32

Data Values¶

0 (disabled)

1 (enabled)

Default Data Value¶

0

Default Allowed¶

False

Request Methods¶

GET, PUT
